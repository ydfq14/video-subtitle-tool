# 智能字幕工坊（Local Subtitle Tool）规格说明书

## 1. 项目概述

**项目名称**：智能字幕工坊（Local Subtitle Tool）
**项目类型**：本地桌面应用（Electron + Vue 3）
**核心定位**：完全在用户本地运行的视频字幕生成工具，为日语/英语视频提供准确的实时中文字幕
**核心价值**：隐私保护（所有处理在本地完成，不上传用户内容）、操作简便、快速生成

## 3. 功能规格

### 3.1 核心功能流程

```
[Step 1: 上传] → [Step 2: 识别与翻译] → [Step 3: 预览与合并]
```

### 3.2 Step 1 - 上传模块

**功能点**：
- 拖拽上传：支持拖拽视频文件到上传区域
- 文件选择：点击按钮打开系统文件对话框
- 文件验证：检查格式（MP4/MOV/AVI/MKV/WebM）和大小（≤2GB）
- 视频预览：上传后自动播放预览

**输入**：
- 视频文件（File对象或拖拽事件）

**输出**：
- 视频文件路径、时长、分辨率、文件大小
- 视频缩略图预览

**错误处理**：
- 不支持的格式 → 显示错误："不支持的视频格式，请上传 MP4、MOV、AVI、MKV 或 WebM 文件"
- 文件过大 → 显示错误："文件大小超过 2GB 限制"
- 读取失败 → 显示错误："视频文件读取失败，请尝试重新上传"

### 3.3 Step 2 - 识别与翻译模块

**功能点**：
- 源语言选择：日语/英语（下拉选择）
- 一键开始处理
- 语音识别（Whisper本地模型）
- 智能断句（基于语义和时长）
- 翻译为中文（本地LLM或API）

**处理流程**：
1. 音频提取（FFmpeg）
2. 语音识别（Whisper）
3. 字幕时间轴同步
4. 翻译处理（可选：直接输出或翻译）
5. SRT字幕文件生成

**实时反馈**：
- 进度百分比（0-100%）
- 当前处理阶段文字
- 详细日志输出

**输出**：
- SRT字幕文件
- 原始识别文本（可选）

**错误处理**：
- 音频提取失败 → 错误代码 E001 + "音频提取失败，请确认视频包含可识别的音频轨道"
- 语音识别失败 → 错误代码 E002 + "语音识别失败，请尝试使用其他视频文件"
- 翻译失败 → 错误代码 E003 + "翻译失败，已生成原始语言字幕，可手动翻译"
- 模型加载失败 → 错误代码 E004 + "AI模型加载失败，请检查网络连接或模型文件"

### 3.4 Step 3 - 预览与合并模块

**功能点**：
- 字幕预览：视频播放器叠加字幕显示
- 字幕编辑：手动调整时间轴和文字
- 一键合成：使用FFmpeg将字幕烧录到视频
- 下载输出：合成后的视频文件

**字幕编辑**：
- 时间轴微调：±100ms 快捷按钮
- 文字修改：直接编辑
- 删除/添加字幕段

**合成设置**：
- 字幕位置：底部居中（默认）/ 自定义
- 字幕样式：字体大小、颜色、描边
- 输出格式：MP4（H.264）

**输出**：
- 带字幕视频文件（MP4）
- SRT字幕文件（可选单独下载）

**错误处理**：
- FFmpeg不可用 → 错误代码 E005 + "视频合成失败，请安装 FFmpeg"
- 合成过程中断 → 错误代码 E006 + "视频合成中断，请重试"

### 3.5 数据结构

```typescript
// 字幕段
interface SubtitleSegment {
  id: number;
  startTime: number;  // 秒
  endTime: number;    // 秒
  originalText: string;
  translatedText: string;
}

// 项目状态
interface ProjectState {
  videoPath: string;
  videoInfo: {
    duration: number;
    width: number;
    height: number;
    size: number;
  };
  sourceLanguage: 'ja' | 'en';
  segments: SubtitleSegment[];
  status: 'idle' | 'uploading' | 'processing' | 'completed' | 'error';
  progress: number;
  logs: LogEntry[];
}

// 日志条目
interface LogEntry {
  timestamp: Date;
  level: 'info' | 'success' | 'warning' | 'error';
  code?: string;
  message: string;
}
```

## 4. 验收标准

### 4.1 功能验收

| 功能 | 验收条件 |
|------|----------|
| 拖拽上传 | 拖拽MP4文件到上传区域后，显示预览并进入Step 2 |
| 文件选择 | 点击按钮可打开系统文件对话框，选择视频后正常预览 |
| 格式验证 | 上传非视频文件时，显示格式错误提示 |
| 源语言选择 | 可切换日语/英语，切换后正确反映在处理逻辑中 |
| 进度显示 | 处理过程中，进度条实时更新，百分比与阶段文字匹配 |
| 日志输出 | 每个处理阶段都有对应日志，时间戳正确 |
| 字幕预览 | 视频播放时，字幕在正确时间显示和消失 |
| 字幕编辑 | 修改字幕文字后，预览同步更新 |
| 视频合成 | 点击合成后，生成带字幕的MP4文件 |
| 下载功能 | 可下载合成后的视频文件和SRT文件 |

### 4.2 视觉验收

- [ ] 界面布局与设计稿一致
- [ ] 颜色使用正确的色值
- [ ] 动画过渡平滑，无卡顿
- [ ] 响应式布局，在不同窗口尺寸下正常显示
- [ ] 错误提示清晰，日志可读

### 4.3 性能要求

- 视频预览：秒开（<1秒）
- 文件验证：<500ms
- 界面交互：<100ms 响应
- 进度更新：实时（每500ms或每个阶段更新）
